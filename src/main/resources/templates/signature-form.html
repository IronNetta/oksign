Comment obtenir le Base64 d'un PDF :

  1. Ouvrez un convertisseur en ligne comme base64encode.org
  2. Uploadez votre fichier PDF
  3. Copiez le résultat Base64 (sans le préfixe "data:application/pdf;base64,")
  4. Collez-le dans le champ ci-dessous
Format : +32 suivi du numéro sans le 0 initial
Incluez l'extension .pdf
Le Base64 doit commencer par quelque chose comme : JVBERi0xLjQK...
Ne pas inclure le préfixe "data:application/pdf;base64,"
Exemple de Base64 valide (début) :
JVBERi0xLjQKJeLjz9MKMSAwIG9iago8PC9UeXBlIC9DYXRhbG9nL1BhZ2VzIDIgMCBSPj4KZW5kb2JqCjIgMCBvYmo8PC9UeXBlIC9QYWdlcy9LaWRzIFs...
